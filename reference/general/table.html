Table classes.

class BaseTable(*arge: object, **kwargs: object)¶

Bases: Table

General table app.

Defines a common style for derived table widgets.

class TableApp¶

Bases: BoxApp

Base table, the container.

In the derived class:

  • Assign source class.

  • Add source urls (list-create, detail).

  • Add table column head names.

  • Add the necessary page widgets.

  • Construct the page DOM.

Important

Override the methods:

  • create_handler()

  • update_handler()

Override the attrs:

  • source_class

  • source_url

  • source_url_detail

  • headings

Variables:

  • source_class – The widget entries data source class.

  • source_url – Url to request the widget entries data.

  • source_url_detail – Url to request the widget entry.

  • headings – The entries table column name.

  • btns_manage – The box of buttons for managing of entries.

  • table – The table of entries list.

  • btns_paginate – The box of buttons for managing of pagination.
